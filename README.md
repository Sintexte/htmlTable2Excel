# htmlTable2Excel
Install Dependencies:
```bash
npm i
```
Help commands:
```bash
node Htmltable2Excelsheet.js -h
```

Command Format:
```bash
node Htmltable2Excelsheet.js -f [htmlfilelocation:REQUIRED] -e [excelfilename] 
```

Example
```bash
node Htmltable2Excelsheet.js -f ./file.html -e Genetics 
```
